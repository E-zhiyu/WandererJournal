# WandererJournal

> 这是一款Material You风格的本地日记APP，界面清爽，功能纯粹，让您随时随地都能记录生活中的点滴。

## ✨特色功能

<table>
<tr>
<td width="50%" style="vertical-align:top">

### 日记标注

支持使用自定义标签标注日记片段，可以用来标注某些有趣的、引人深思的、反感的或其他能够带来不同感觉的片段。

</td>
<td width="50%" style="vertical-align:top">

### 日记关键词

有时候遇到想记录的事情，但是手头上又很忙无法记日记时，可以输入一个日记关键词，等到有空再写日记时就能提醒自己防止遗漏。

</td>
</tr>
<tr>
<td width="50%">

### 日记分享

可以选择分享某一段日记内容，支持导出为多种格式（图片、Markdown文档等等）。

</td>
<td width="50%">

### 数据管理与隐私保护

用户产生的所有数据均保存在本地，并且内置的数据导入导出功能。为了防止他人查看日记内容，APP内置了身份验证功能，开启后在进入APP时需要进行身份验证。

</td>
</tr>
<tr>
<td width="50%">

### 日记提醒

用户可以设置提醒时间，如果到了提醒时间还没有记日记，APP会发送通知提醒用户写日记。

</td>
</tr>
</table>

## 🛡️权限申请

以下为APP需要申请的权限，这里只做简单说明，有关各个功能需要什么权限请在APP内部查看。

| 权限名称         | 说明                               |
|--------------|----------------------------------|
| **通知权限**     | 通过发送应用通知提醒用户                     |
| **精确闹钟权限**   | 用于自动触发某些定时任务                     |
| **系统启动监听权限** | 开机时启动定时任务的自动安排，防止设备关机后无法正常执行定时任务 |

## 📡作者的话

软件的名称参考了《空洞骑士》中的收集品“漫游者日记”。作者认为每个人都在以自己的方式寻找人生的价值，就像漫游者一样经历各种“冒险”，因此有这么一个“漫游者日记”记录生活再合适不过了。

本人从高三开始记日记，自从大学学了安卓开发后一直想做一个纯粹干净又好用的日记软件，但由于时间问题一直没有开始，直到大二下学期期中需要Java答辩，作者不想把已经很完善的记账APP浪费在期中答辩上，所以启动了此项目。
