# WandererJournal

> 这是一款Material You风格的本地日记APP，界面清爽，功能纯粹，让您随时随地都能记录生活中的点滴。

## ✨特色功能

<table>
<tr>
<td width="50%" style="vertical-align:top">

### 情绪标注

使用自定义情绪标签标注日记片段，让您回顾日记时能更加了解当时的心情。

</td>
<td width="50%" style="vertical-align:top">

### 数据统计

在应用主页会显示简略的日记数据统计信息，点击“日记数量”或“段落数量”后跳转至详细统计数据界面，里面记录了所有日记的统计数据，并设置有“记忆像素”（类似于GitHub贡献墙）便于查看近一年的日记情况。

</td>
</tr>
<tr>
<td width="50%" style="vertical-align:top">

### 内容搜索

在读日记时可以搜索段落内容和标注的情绪标签，并且能够在搜索出的结果之间快速跳转。

</td>
<td width="50%" style="vertical-align:top">

### 日记关键词

当遇到想记录的事情但手头上又很忙时，可以输入一个日记关键词，等到有空再写日记时就能提醒自己防止遗漏。

</td>
</tr>
<tr>
<td width="50%">

### 日记分享

选择一段日记片段并将其分享给其他人。

</td>
<td width="50%">

### 数据管理与隐私保护

所有数据均保存在本地，支持导出数据为备份文件或者从备份文件中恢复数据。内置身份验证功能，开启后在进入APP时需要进行身份验证。

</td>
</tr>
<tr>
<td width="50%">

### 段落追加和日记导入

从文本文件中追加日记段落，段落的时间与文件修改时间一致，或者从文本文件中导入整个日记（需要用单独的一行区分不同日期的日记内容）。

</td>
</tr>
</table>

## 📡作者的话

软件的名称参考了《空洞骑士》中的收集品“漫游者日记”。我觉得每个人都在以自己的方式寻找人生的价值，就像漫游者一样经历各种“冒险”，因此有这么一个“漫游者日记”记录生活再合适不过了。

我从高三开始记日记，自从大学学习安卓开发后一直想做一个纯粹干净又好用的日记软件，但由于时间问题一直没有开始，直到大二下学期期中需要Java答辩，作者不想把已经很完善的记账APP浪费在期中答辩上，所以启动了此项目。
