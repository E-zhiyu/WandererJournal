# Pure Diary

> 这是一款Material You风格的本地日记APP，界面清爽，功能纯粹，让您随时随地都能记录生活中的点滴。

## ✨特色功能

<table>
<tr>
<td width="50%" style="vertical-align:top">

### 日记保存

按照日期由近到远的顺序保存日记内容，支持使用自定义标签标注写下某段内容时的情绪。软件还会自动统计日记的第一天与今天的间隔天数、写了日记的天数等数据并显示给用户。

</td>
<td width="50%" style="vertical-align:top">

### 快速日记

输入一段文字后点击保存按钮，输入的内容将自动追加至当天的日记内容之后，就像使用通讯软件发送消息一样。

</td>
</tr>
<tr>
<td width="50%" style="vertical-align:top">

### 日记分享

可以选择分享某一段日记内容，支持导出为多种格式（图片、MD文档等等）。

</td>
<td width="50%" style="vertical-align:top">

### 数据管理与隐私保护

用户产生的所有数据均保存在本地，用户可以随时利用APP内置的数据导出功能备份数据。本APP还内置身份验证功能，启用时每次进入APP都将要求用户进行身份验证。

</td>
</tr>
