# v1.0.0

> 该版本为首个Release版本，以下为核心功能概述

### 日记管理

- 在写日记界面以聊天软件的方式记日记
- 在日记界面，保存的日记项会按照日期由近到远排序
- 所有保存的日记段落都能在读日记界面中查看，并且以时间由远到近的顺序排序
- 在读日记和写日记界面可以对日记段落进行编辑

### 情绪标记

- 支持自定义情绪标签
- 在读日记和写日记界面中可以为日记段落绑定多个情绪标签，并为每个情绪标签单独设置强烈等级
- 绑定的情绪标签会出现在对应日记段落文本内容的下方

### 日记搜索

- 支持搜索日记段落内容以及情绪标签
- 搜索成功后能够在多个搜索结果之间进行跳转
- 搜索成功后会对搜索的关键词或者情绪标签进行高亮处理，未匹配搜索结果的日记段落则正常显示

### 媒体保存

- 支持为日记段落添加多个媒体文件
- 添加的媒体文件会显示在日记段落文本内容的上方
- 点击显示的媒体文件缩略图能够查看媒体文件大图

### 数据管理

- 所有数据均保存在本地
- 支持将数据导出为备份文件，且支持从备份文件中恢复数据
- 支持从外部文本文件中追加日记段落，追加的段落的时间与文本文件最后编辑时间一致
- 支持从文本文件导入整个日记，前提是该文本文件用单独的日期行隔开了不同日期的日记内容

### 隐私保护

- 内置身份验证模块，启用和关闭都需要验证身份
- 启用身份验证功能后，每次进入应用都将触发身份验证逻辑（进入软件时与上次身份验证间隔一定时间才会要求验证身份），支持调整两次身份验证的最小时间间隔
- 身份验证的方式取决于设备的屏幕解锁方式
